# GopSalt CC Lua Library

Коллекция скриптов для ComputerCraft в Minecraft (ATM10 и другие сборки).

## Доступные скрипты и программы

| Название | Папка | Описание | Код Pastebin | Команда для установки |
| :--- | :--- | :--- | :--- | :--- |
| **Patrick Star Crafter** | [patrick/](./patrick) | Автокрафт Звезды Патрика в ATM10 (Create + AE2) | `DKyy7aEK` | `pastebin get DKyy7aEK startup` |

## Как использовать
Все скрипты снабжены инсталлятором для удобной загрузки на сервере:
1. Скопируйте команду установки для нужной программы из таблицы выше.
2. Введите её в консоль черепашки/компьютера в Minecraft.
3. Скачанный файл будет сохранен как `startup` и автоматически обновит и запустит основной код при каждом запуске черепашки. Установщик сам выкачает актуальные скрипты с GitHub.
